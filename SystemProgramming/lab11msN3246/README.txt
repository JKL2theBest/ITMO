lab11msN3246 - Работа с файлами и каталогами

Описание
Программа для поиска заданной последовательности байтов в файлах, начиная с указанного каталога. Реализован рекурсивный обход каталогов с помощью функции nftw(). Программа поддерживает вывод справки, версии (информацию о студенте) и отладочную информацию через переменные окружения.

Сборка
Убедитесь, что у вас установлен компилятор gcc.

Для сборки программы выполните команду:
make

Для очистки сгенерированных файлов используйте:
make clean

Запуск
Программа запускается следующим образом:

./lab11msN3246 [опции] <каталог> <цель_поиска>

Примеры:
Для поиска последовательности байтов 0xc0ffee в каталоге /home:
./lab11msN3246 /home 0xc0ffee

Для вывода справки:
./lab11msN3246 --help

Для вывода информации о версии:
./lab11msN3246 --version

Опции:
-h, --help - выводит справку по использованию.
-v, --version - выводит информацию о студенте.

Переменные окружения
LAB11DEBUG - Включает режим отладки. При его установке программа выводит отладочную информацию о процессе поиска.

Логирование ошибок
Ошибки, такие как проблемы с доступом к файлам, записываются в системный журнал с помощью функции syslog().
