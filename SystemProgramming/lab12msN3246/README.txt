Для сборки проекта (вместе с плагином) используйте следующую команду:

	make или make all

## Запуск

Для запуска программы используйте следующую команду:

	./lab12msN3246 [опции] [каталог_поиска]

### Замечание:

Аргументы для опций плагинов должны передаваться через знак `=`, а не через пробел. Например:

	--entropy=1
